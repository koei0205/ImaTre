# いまトレ

## 概要

「いまトレ」は、日常の中で短時間の運動を習慣化することを目的としたAndroidアプリです。
大学の授業「プログラミング実践Ⅱ」の最終課題として、Androidアプリ開発の基本的な構成や実装方法を学ぶために制作しました。

本アプリでは、ユーザーに運動ミッションを提示し、達成状況や運動記録を管理できる仕組みを実装しています。
現在は、アプリとしての基本機能を優先して実装しており、UIやデザイン面については今後改善予定です。

## 制作目的

このアプリは、Androidアプリ開発の基礎を実践的に学ぶことを目的として制作しました。

特に、以下のような内容を実装しながら学ぶことを意識しています。

* Javaを用いたAndroidアプリ開発
* XMLによる画面レイアウトの作成
* ActivityとFragmentを用いた画面構成
* BottomNavigationViewによる画面切り替え
* アプリ内での状態管理
* ローカルデータの保存
* Git / GitHubを用いたバージョン管理
* Android Studioでのビルド・動作確認

単に画面を作成するだけではなく、ユーザーの操作に応じて状態が変化し、記録が残るアプリを作ることを目標にしました。

また、「運動を記録するアプリ」ではなく、「今すぐ少しだけ運動するきっかけを作るアプリ」というコンセプトで設計しています。
日常生活の中で、短時間の運動を自然に取り入れられるようにすることを目指しました。

## 主な機能

アプリでは、スクワットや腕立て伏せなどの運動ミッションを提示し、ユーザーが30分以内に達成できたかどうかを記録します。
また、1日のミッション回数制限、達成状況の保存、推定消費カロリーの表示、過去の履歴確認などの機能を実装しています。

## アプリ構成

本アプリは、BottomNavigationViewを用いて複数の画面を切り替える構成にしています。

主な画面は以下の通りです。

* ホーム画面
* ミッション画面
* 履歴画面

各画面はFragmentとして分けて実装し、MainActivityから画面を切り替える構成にしています。
これにより、画面ごとの役割を分離し、処理を整理しやすい形にしました。

## 現在の状況

現在は、授業提出に向けてアプリの基本機能を実装している段階です。

そのため、現時点では授業提出用・動作確認用としての完成度を優先しており、一般公開できる状態にはまだ至っていません。
